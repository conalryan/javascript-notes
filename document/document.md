# [Document](https://developer.mozilla.org/en-US/docs/Web/API/Document)

- Interface that represents any web page loaded in the browser and serves as an entry point into the web page's contnet, which is the **DOM tree**.
- Provides functionality globally to the document, like how to obtain the page's URL and create new elements in the doucment.
- Describes common properties and methods for any kind of document.
- Depending on the document's type (e.g. HTML, XML, SVG, ...) a larger/smaller API is available.
- HTML docuemtns served with "text/html" content type implent the HTMLDocument interface.
- XML and SVG documents implement XMLDocument interface.

EventTarget <- Node <- Document

- When member returns an object of type document (e.g. ownerDocument property of an element returns the document to which it belongs), this object is the root document itself.

## Element

- Element or node of type element returned by member of the DOM API.
- Example: document.createElement() method returns an object reference to a node, however we say it returns an element.
-

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <title>document-inf</title>
  </head>
  <body>
    <h1>Document Inf</h1>
    <script src="document-inf.js"></script>
  </body>
</html>
```
